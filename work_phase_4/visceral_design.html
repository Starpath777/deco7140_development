Visceral Design

The focus of this section of the Style Guide is ensuring the user has a good first impression of the site.
本节风格指南聚焦于打造用户对网站的第一视觉良好体验。

Reference diagram

Style Guidelines / 风格指南

Typography / 排版规则

  • Font Family 字体家族: 'Exo', serif
  • Font Sizes 字号: h1:3rem, h2:2rem, p:1rem
  • Font Weight 字重: normal:400, bold:700
  • Line Height 行高: 1.6
  • Letter Spacing 字距: 0.5px

标题示例 H1

这是一段文本示例，用于展示排版效果。This is a paragraph example to demonstrate typography.

h1 { font-family: 'Exo', serif; font-size: 3rem; font-weight:700; line-height:1.6; letter-spacing:0.5px; }

Color Scheme / 配色方案

  • Primary 主色: var(--color-primary) (#0ff0fc)
  • Secondary 次色: var(--secondary-color) (#3498db)
  • Accent 强调色: var(--color-accent) (#ff007f)

对比度 Contrast Ratios: Primary on background ≥ 4.5:1

Layout / 布局规范

  • Container 容器宽度: max-width:1200px; margin:0 auto;
  • Spacing 间距变量: --space-1:0.5rem; --space-2:1rem; --space-3:2rem;
  • Breakpoints 响应式断点: 768px, 1024px

Image Guidelines / 图像准则

  • Aspect Ratio 尺寸比例: 16:9
  • Alt Text 替代文本: 必填且描述清晰
  • Icons 图标风格: 使用高对比度 SVG 矢量
