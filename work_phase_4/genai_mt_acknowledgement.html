GenAI & MT Acknowledgement

GenAI/MT 使用清单 / Usage Checklist

  1. Prompt 设计 / Prompt Design：
    编写明确指令，例如“生成站点地图页面的 HTML 结构”。
    Crafted clear prompts such as “Generate the HTML skeleton for the site map page.”
  2. 生成草稿 / Draft Generation：
    使用 ChatGPT 生成初稿，涵盖内容结构与示例。
    Used ChatGPT to generate initial drafts covering structure and sample content.
  3. 人工审核 / Human Review：
    检查输出的准确性、风格一致性与可访问性要点。
    Reviewed for accuracy, style consistency, and accessibility compliance.
  4. 修改与优化 / Edits & Refinements：
    根据反馈修改文本与代码，确保符合项目需求。
    Refined text and code based on feedback to meet project requirements.
  5. 引用与声明 / Citation & Acknowledgement：
    在页面注明使用 GenAI/MT 步骤和框架要点。
    Documented the use of GenAI/MT steps and key framework points in the page.
  6. 机器翻译校对 / MT Proofreading：
    使用机器翻译生成双语内容，并人工校对术语与流畅度。
    Used machine translation for bilingual content and manually proofread terminology and flow.

反思与评估 / Reflection & Evaluation

站点地图阶段 / Site Map Stage

帮助 / Benefits: 快速生成页面框架与导航结构示例，节省初始布局时间。
Provided rapid scaffolding of page structure and navigation examples, saving time on initial layout.
局限 / Limitations: 自动生成的 HTML 需要针对项目 CSS 做适配和调优。
Auto-generated HTML required manual adaptation and tweaks for project-specific CSS.
改进 / Improvements: 可在 prompt 中加入更具体的 CSS 类与组件规范。
Include more detailed CSS classes and component guidelines in prompts.

风格指南阶段 / Style Guide Stage

帮助 / Benefits: 提供一致的色彩与排版建议，增强视觉连贯性。
Offered consistent color and typography suggestions, enhancing visual coherence.
局限 / Limitations: 生成的配色方案偶有与品牌基调不符情况。
Auto-generated palettes sometimes misalign with brand tone.
改进 / Improvements: 在提示中附上品牌现有调色板与示例，确保风格一致。
Attach existing brand palette and examples in the prompt to ensure style alignment.

伦理回应阶段 / Ethics Response Stage

帮助 / Benefits: 生成 ACS 伦理准则引用与潜在风险分析框架。
Generated ACS Code of Ethics citations and risk analysis framework.
局限 / Limitations: 分析深度有限，缺少具体风险度量与缓解方案。
Ethics analysis lacked depth, missing concrete metrics and mitigation strategies.
改进 / Improvements: 在提示中明确要求“量化风险”并提供缓解示例。
Specify “quantify risks” in prompts and provide mitigation examples.
